### Beertrader

Welcome to my app.

It is a beer trading app that allows users to connect and trade all kinds of beer.

In this app, users will be able to login to view potential beers for trade. 

### Technologies Used

HTML  |  CSS  |  Javascrip  |  Node.js  |  Mongo  |  Mongoose
